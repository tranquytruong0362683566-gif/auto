FB Post Title Reader

Nhập UID/link bài viết Facebook. Kết quả chỉ hiển thị tiêu đề bài viết, không hiện link, tên nhóm, thời gian hay chữ Facebook.

Nhập dữ liệu
Mở extension Bridge → copy Extension ID → dán vào đây. Web GitHub Pages cần ID này để gửi lệnh cho extension.
Có thể nhập nhiều dòng. Nếu chỉ nhập số UID, web tự ghép thành https://www.facebook.com/UID.
Nên để 4000–7000 nếu bài có ảnh/video.
Số ký tự tối đa của tiêu đề hiển thị.
